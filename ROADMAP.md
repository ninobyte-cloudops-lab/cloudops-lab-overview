# Roadmap

A high-level view of how the AI-Native CloudOps Lab matures. This is an overview only; no internal dates or implementation details are published.

## Phases

### 1. Product architecture
Define the lab's learning outcomes, sandbox model, and governance approach.

### 2. Student onboarding
Design a clear, low-friction path for learners to enter the governed sandbox safely.

### 3. Sandbox validation
Confirm the controlled environment behaves predictably and keeps practice contained.

### 4. Cost and safety validation
Validate the cost and safety gates before any live AWS execution is enabled.

### 5. Pilot cohort readiness
Prepare a small, reviewed pilot once the prior phases are validated.

---

Progression between phases is gated by readiness, not by calendar. Live AWS execution remains gated until cost and safety validation is complete.
